# ai-writing-reaserch
Exploring AI-assisted workflows for academic writing, vibe coding, and research experiments.

## 📌 关于本仓库
探索人工智能在学术写作中的辅助应用，聚焦硕士大论文精炼为可发表学术小论文的实践，同时涵盖 Prompt 工程与相关资源整理。

本仓库旨在记录学习与探索过程、分享经验，既帮助自己系统化探索 AI 工具，也为其他学术研究者提供参考和借鉴，并欢迎学术研究者交流与讨论。

## 🔹 仓库内容概览

本仓库内容分为三部分：

1. **学位大论文精炼为学术小论文的 AI 实践探索**  
   记录从大论文到小论文的完整实践过程，包括内容拆解、AI 辅助生成与润色尝试、遇到的问题与失败案例，以及总结出的有效策略与经验。

2. **Prompt 研究与整理**  
   总结在实践探索中形成的高效学术写作 Prompt，包含 Prompt 设计思路、优化方法，以及提高 AI 输出逻辑性和专业性的技巧。

3. **相关资源整理**  
   收集和整理与 AI 辅助学术写作相关的工具、文档和学习材料，为实践和 Prompt 研究提供参考和支持。
